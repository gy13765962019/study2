    4. 热点结构：`<a-entity>` 容器 → 3 子元素（中心箭头 + 双层扩散环），全部 `look-at="[camera]"` 永远朝相机

- **Hotspot 第二轮修复（09:18）·彻底解决黑块 + 动态箭头**：
  - 用户反馈："重新检查，那三个前进按钮，1、我要的指向前进的箭头样式，2、我要的是动态的"
  - 根因复盘（之前为什么还是有问题）：
    1. `look-at` 是 A-Frame **Extras** 插件组件，不是 core 内置；之前没引入 `aframe-extras` 脚本，导致 `look-at` 属性被静默忽略，热点子元素朝向随机，可能呈现为细线或方块
    2. `<a-animation>` 是 A-Frame 0.x 时代的子元素语法，在 1.4.2 中仍兼容但常因 DOM 操作时序问题导致动画不启动 → 实体凝固在初始 scale 0.5 状态
    3. `forward.svg` 设计过大（直径占 256px），中央箭头被淹没
  - 修复（commit 9417ba0）：
    1. **加 aframe-extras 脚本** `<script src="https://unpkg.com/aframe-extras@7.5.0/dist/aframe-extras.min.js"></script>`，让 `look-at="[camera]"` 真正生效
    2. **主图标改为 PNG** `assets/hotspot/forward.png`：256×256 透明背景 + 中心向上大箭头（青色+白边）+ 深色透明底盘 + 中心白点 + 外圈青色描边
    3. **动画改用现代 A-Frame 内置 `animation` 组件**（命名后缀支持多动画叠加）：
       - `animation__scale`：scale 0.9↔1.15 呼吸
       - `animation__bob`：position 0→0.12 上下浮动
       - `animation__fade`：opacity 0.9→0 淡出
    4. **三层动态**叠加确保"明确动态"：
       - 外层柔光环：scale 0.6→2.2 + opacity 0.6→0，dur 2400ms
       - 扩散环 1（青）：scale 0.5→1.8 + opacity 0.9→0，dur 2000ms
       - 扩散环 2（绿）：同上但 delay 1000ms 错峰
       - 中心箭头：scale 0.9↔1.15 + position bob 0→0.12，dur 1400ms alternate
    5. **防黑边**：所有 material 加 `depthWrite: false` + `alphaTest: 0.01`，避免半透明边缘被深度测试误判
    6. 删除废弃的 `forward.svg` 和未使用的 `ring.png` / `ring2.png`，只保留 `forward.png`
    7. 新增 `scripts/gen_hotspots.py` 记录图标生成方法（可重复跑）
  - 验证：CDN 刷新后 `forward.png` → 200 (2493 bytes) / `forward.svg` → 404（确认已下线）/ index.html 包含 `aframe-extras` + 11 处 `animation__` 模式
